# Утилита фильтрации содержимого файлов

## Описание

Консольная утилита для фильтрации содержимого текстовых файлов.

При запуске утилиты в командной строке передаётся несколько входных файлов, содержащих вперемешку:

* целые числа;
* вещественные числа;
* строки.

Каждая строка во входных файлах рассматривается как отдельный элемент.
Утилита читает файлы последовательно в порядке их перечисления в командной строке и распределяет данные по типам.

Результат работы — запись данных разных типов в отдельные выходные файлы и вывод статистики в консоль.

---

## Требования

* **Java 21 (LTS)**
* **Apache Maven** (для сборки проекта)

Сторонние библиотеки не используются.

---

## Сборка проекта

Для сборки проекта выполните команду в корне проекта:

```bash
mvn clean package
```

После успешной сборки будет создан JAR-файл:

```text
target/filefilter-1.0.0.jar
```

---

## Запуск программы

Общий вид команды запуска:

```bash
java -jar target/filefilter-1.0.0.jar [опции] файлы
```

### Пример запуска

```bash
java -jar target/filefilter-1.0.0.jar -a -o out -p result_ in1.txt in2.txt
```

---

## Поддерживаемые опции

| Опция         | Описание                                                                                       |
| ------------- | ---------------------------------------------------------------------------------------------- |
| `-a`          | Добавление данных в существующие выходные файлы (append). По умолчанию файлы перезаписываются. |
| `-o <path>`   | Путь к папке для выходных файлов. По умолчанию используется текущая директория.                |
| `-p <prefix>` | Префикс имён выходных файлов.                                                                  |
| `-s`          | Краткая статистика.                                                                            |
| `-f`          | Полная статистика.                                                                             |

---

## Выходные файлы

В зависимости от найденных данных создаются следующие файлы:

* `integers.txt` — целые числа
* `floats.txt` — вещественные числа
* `strings.txt` — строки

Если указан префикс (`-p`) и/или путь (`-o`), имена файлов изменяются соответствующим образом.

**Файлы создаются только при наличии данных соответствующего типа.**

---

## Статистика

После обработки всех файлов программа выводит статистику в консоль.

### Краткая статистика (`-s`)

* количество целых чисел;
* количество вещественных чисел;
* количество строк.

### Полная статистика (`-f`)

Дополнительно:

**Для чисел:**

* минимальное значение;
* максимальное значение;
* сумма;
* среднее значение.

**Для строк:**

* длина самой короткой строки;
* длина самой длинной строки.

---

## Обработка ошибок

* Все возможные ошибки ввода-вывода обрабатываются.
* Программа не завершается аварийно при ошибках чтения отдельных файлов.
* При невозможности продолжения работы пользователю выводится сообщение с указанием причины.
